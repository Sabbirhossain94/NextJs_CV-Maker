## Installation

1. Clone down this repository. You will need node and npm installed globally on your machine.

```bash
  $ git clone https://github.com/Sabbirhossain94/NextJs_CV-Maker.git

```
2. Install:
```bash
  $ npm install 

```
3. To Start Server: 

```bash
  $ npm run start/dev 

```
